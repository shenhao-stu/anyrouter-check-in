🔄 AnyRouter Cookie Updater

GitHub 配置
需要 repo 权限（或 Environment secrets 写入权限）
留空则推送到 repository secrets
账号配置
⚠ JSON 格式错误
domain（必填）· api_user（留空自动解析）· env_key_suffix（留空自动生成为 {api_user}_{PROVIDER}）· cookie_name（留空默认 session）
定时设置
建议 360（6 小时），与签到频率一致
⚡ 工作原理：从浏览器提取 session cookie → 调 /api/user/self 自动获取 api_user → GitHub API 加密推送为 ANYROUTER_ACCOUNT_* Secret → 签到脚本自动读取最新 cookie
💡 Heibai：自动提取多个认证 cookie（无需手动配置 cookie_name）
⚠️ 前提：保持各站点登录状态
